4.  Slider :

Slider is a sliding div, which can be horizontal or vertical.

To show any products or banner in your web page, sliders are used.

Some examples are shown below :

Value: 

                
    <div class="slidecontainer" id="btn-link">
        <input type="range" min="1" max="100" value="50" class="slider" id="range">
        <p class="content"> Value: <span id="sliderValue"></span></p>
    </div>
                
            
                
    <script>
        let slider = document.getElementById("range");
        let output = document.getElementById("sliderValue");
        output.innerHTML = slider.value;
        slider.oninput = function()
        {
            output.innerHTML = this.value;
        }
    </script>
