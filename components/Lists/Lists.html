8.  List :

Lists are used to specify lists of information. All lists may contain one or more list elements.

Some examples are shown below:

    Games
  • Gta V
  • Valorant
  • Cs : Go
  • Max Payne Triology
                
        <ul class="lists">

        <div class="list-title">Games</div>

        <hr>

        <li>Gta V</li>

        <li>Valorant</li>

        <li>Cs : Go</li>

        <li>Max Payne Triology</li>

    </ul>

List items can also contain sub-list items.

Some examples are shown below :

    Categories
  • Mobile
    • Gaming Laptops
    • Legion
    • Omen
    • Ideapad Gaming
    • Alienware
    • Predator
  • Consoles
  • Accessories
            
    <ul class="lists">

        <div class="list-title">Categories</div>

        <hr>

        <li>Mobile</li>

        <li>

            <ul>

                <div class="-sub-list-title">Gaming Laptops</div>

                <li>Legion</li>

                <li>Omen</li>

                <li>Ideapad Gaming</li>

                <li>Alienware</li>

                <li>Predator</li>

            </ul>

        </li>

        <li>Consoles</li>

        <li>Accessories</li>
        
    </ul>
