4.  Button :

Buttons are primarily used to "do something" on a website.

If the action is to created, edit, delete or anything else to some piece of information, use a button.

Some examples are shown below:

                
    <button class="btn">Default</button>

    <button class="btn-primary">Primary</button>

    <button class="btn-success">Success</button>

    <button class="btn-error">Error</button>

Buttons with outline.

Some examples are shown below :

            
    <button class="btn-outline">Default</button>

    <button class="btn-primary-outline">Primary</button>

    <button class="btn-success-outline">Success</button>

    <button class="btn-error-outline">Error</button>

Buttons with Link.

Some examples are shown below :

                
    <button class="btn""><a class="btn-link"href="">Link</a></button>
