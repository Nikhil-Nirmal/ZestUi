1.  Alert :

Alert messages can be used to notify the user about something special: danger, success, information or warning.

Some examples are shown below:

Error! This is an Error Alert.

Warning! This is a Warning Alert.

Primary! This is a Primary Alert.

Success! This is a Success Alert. 

                
    <div class="errorAlert">
        <h1 class="errorAlertText">
        Error! This is an Error Alert. 
        </h1>
    </div>

    <div class="warningAlert">
        <h1 class="warningAlertText">
        Warning! This is a Warning Alert.</h1>
    </div>
    
    <div class="primaryAlert">
        <h1 class="primaryAlertText">
        Primary! This is a Primary Alert.</h1>
    </div>
                    
    <div class="successAlert">
        <h1 class="successAlertText">
        Success! This is a Success Alert. </h1>
    </div>
