1.  Avatar :

Avatars are images that users can set as their profile picture.

Some examples are shown below:

Avatar's size can be costumised upto one's preferences.

Some examples are shown below :

            
    <img class="avatar-sm" src="../../Images/avatar1.jpg"  alt="avatar">

    <img class="avatar-md" src="../../Images/avatar2.png"  alt="avatar">

    <img class="avatar-lg" src="../../Images/avatar3.jpg"  alt="avatar">

Avatar's can be of square shape.

Just add -square in the end

Some examples are shown below :

        
    <img class="avatar-sm-square" src="../../Images/avatar1.jpg"  alt="avatar">

    <img class="avatar-md-square" src="../../Images/avatar2.png"  alt="avatar">

    <img class="avatar-lg-square" src="../../Images/avatar3.jpg"  alt="avatar">
