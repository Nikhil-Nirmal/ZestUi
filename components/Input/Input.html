7.  Inputs :

Inputs are used in forms to collect user related data.

There are many methods to collect data from user.

Some examples are shown below:

                
    <label for="input-username">username*

        <input type="text" class="input">

     </label>

    <label for="input-password">password*

        <input type="password" class="password">
                        
    </label>

Input field for First name and last name

Some examples are shown below :

            
    <label for="text" id="btn-link"> <span class="input-text"> First-Name & Last-Name : </span> <input class="FN" type="text"> <input class="LN" type="text"> </label>

Radio buttons allow the user to select one option from a given set.

Some examples are shown below :

        
    <label for="checked"> <input class="unchecked-input" type="radio" name="component" checked> Checked</label>\

    <label for="Unchecked"> <input class="normal-input" type="radio" name="component"> Unchecked</label>

    <label for="disabled" "> <input class="disabled-input" type="radio" disabled> Disabled</label>

Checkbox allows a user to choose multiple option from a given set.

Some examples are shown below :

        
    <label for="checked"> <input class="unchecked-input" type="checkbox" name="component" checked> Checked</label>
    
    <label for="Unchecked"> <input class="normal-input" type="checkbox" name="component"> Unchecked</label>

    <label for="disabled" > <input class="disabled-input" type="checkbox" disabled> Disabled</label>
