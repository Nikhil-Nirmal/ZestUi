Getting Started :

Installing and setting up Zest UI for your usage.

How to install :

Copy-paste the stylesheet <link> into your <head> to load our CSS.

                     <link rel="stylesheet" href="https://zest-ui.netlify.app/Css/Component.css">

Starter Template :

After finishing the setup your code should look something like this....

                
     <!DOCTYPE html>

     <html lang="en">
                    
     <head>

     <meta charset="UTF-8">

         <meta http-equiv="X-UA-Compatible" content="IE=edge">

             <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    
             <!-- Zest UI-->

             <link rel="stylesheet" href="https://zest-ui.netlify.app/Css/Component.css" />
                    
             <title>Zest UI</title>
                        
             </head>
                    
             <body>
                    
             <h1>Zest UI here!!!</h1>
                    
         </body>
                    
     </html>
